WaveShaper : Table lookup waveshaping module¶

Description¶

This module applies a waveshaping-based distortion on the input sound. It allows the user to draw the transfert function on the screen.

Sliders¶

Filter Freq :

Center frequency of the post-process filter

Filter Q :

Q factor of the post-process filter

Graph Only¶

Overall Amplitude :

The amplitude curve applied on the total duration of the performance

Transfer Function :

Table used as transfert function for waveshaping

Popups & Toggles¶

Filter Type :

Type of the post-process filter

Polyphony Voices :

Number of voices played simultaneously (polyphony), only available at initialization time

Polyphony Spread :

Pitch variation between voices (chorus), only available at initialization time
